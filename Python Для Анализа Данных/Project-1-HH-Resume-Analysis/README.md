## Описание проекта
Цель данного проекта — провести анализ данных о соискателях с платформы HeadHunter и подготовить их для построения модели, которая сможет автоматически определять примерный уровень заработной платы, подходящей пользователю, исходя из информации, указанной им в резюме.

Перед построением модели необходимо выполнить предварительный анализ, преобразование, разведывательный анализ и очистку данных.

## Цель и задачи
Наша задача — подготовить данные в таблице о соискателях, чтобы на их основе в дальнейшем можно было построить модель прогнозирования заработной платы.

Наша цель — провести анализ и очистку данных о соискателях, выявить ключевые факторы, влияющие на уровень заработной платы, и подготовить таблицу для последующего построения модели прогнозирования

### Метрика качества
На данном этапе не оценивается качество модели — внимание уделяется качеству данных, их корректности и пригодности для дальнейшего анализа.

### Что мы отрабатываем
- Владение инструментами анализа данных (**pandas, numpy**).
- Навыки визуализации (**matplotlib, seaborn, plotly**)
- Предобработка, очистка и исследование реальных данных

## Описание данных
Для анализа используются реальные данные о соискателях с сайта HeadHunter.
В датасете содержится информация о:

- Пол, возраст — основные демографические данные соискателя;
- ЗП — уровень желаемой заработной платы;
- Ищет работу на должность — профессия или позиция, на которую претендует кандидат;
- Город, переезд, командировки — место проживания и готовность к переезду или командировкам;
- Занятость — предпочтительный формат занятости (полная, частичная, проектная и т. д.);
- График — желаемый режим работы (офис, гибкий график, удалёнка и т. д.);
- Опыт работы — суммарный трудовой стаж в годах и месяцах;
- Последнее/нынешнее место работы — организация, где кандидат работал последним;
- Последняя/нынешняя должность — последняя занимаемая должность;
- Образование и ВУЗ — уровень образования и учебное заведение;
- Обновление резюме — дата последнего обновления данных кандидатом;
- Авто — наличие личного автомобиля.

Перед исследованием данные проходят этап загрузки, проверки и предварительного анализа.

## Инструкция по запуску проекта
1. Склонируйте репозиторий:

``git clone https://github.com/paginxd/SF-ML-Homeworks.git``


2. Перейдите в папку проекта:

``cd Сессионое задание``


3. Установите необходимые библиотеки:

``pip install -r requirements.txt``


4. Подготовка данных:

Датасет слишком большой для GitHub (>450 Мб), поэтому файл с данными добавлен в ``.gitignore``.

Скачать данные можно по ссылке: ``https://disk.yandex.ru/d/ArpSlnOuoLTo3A``

После скачивания положите файл с данными в проект во внутреннюю папку: ``data/``.

5. Запуск проекта:   

Откройте ваш проектный ноутбук.  

Выполняйте ячейки по порядку.  

6. Визуализации и Plotly:

Интерактивные графики Plotly сохранены в формате HTML в папке ``plotly_graphics/``.
